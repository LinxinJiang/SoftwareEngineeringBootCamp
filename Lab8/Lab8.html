MyIcon

What exactly is a favicon?

Favicon is the webiste icon. It's the logo of the web. A good favicon can make people rembember your webside.



How are favicons created?

Use Pixelation. Set to a suitable image width,height, and bits per pixel. Then use binary/ Hexadecimal to create.



How did I create the favicon for this site?

Set image width & height to 17, bits per pixel to 18. Then be creative!
